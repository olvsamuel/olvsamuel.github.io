O que é HTML?

HTML - Hypertext Markup Language - é uma linguagem de marcação para apresentar conteúdos na web, tudo isso por tags.

Já o HTML5, é a mais recente versão dessa linguagem. Possuí novos elementos e novas formas de utilização, comparado a versões anteriores, como o HTML4.


Estrutura básica do HTML

<!DOCTYPE html> - Indica para o navegador, que voce está utilizando HTML5

<html lang="pt-br"> - Abre a tag HTML, no qual dentro estará a estrutura do site. Lang indica a linguagem do texto do site.

<head> - Abre a tag HEAD, dentro ficaram informações sobre a página, os metadados.

<meta charset="utf-8"> - CHARSET indica o tipo de caracteres utilizados no seu site, para o Brasil, utilizamos utf-8

<title> Título da Página </title> - Sem muito segredo, aqui ficará o título da página

</head> - Fecha o HEAD, não esquecendo da /

<body> - Abre a tag BODY, aqui ficará todo o conteúdo do site, ou seja, tudo que ficará visivel para o usuário

</body> - Fecha o BODY, não esquecendo da /

</html> - Fecha o HTML, não esquecendo da /


Iniciando um projeto

Para iniciar um projeto HTML, será necessário um editor de texto.
Os mais usados, são:

  • Sublime Text

    Um dos mais famosos, simples e práticos de usar.
    É um editor de texto, para html é bem completo.

  • Atom

    Recente, 2015, mas se mostrou estável
    É um editor de texto desenvolvido pelo GitHub, permite também o uso de extensões para Node.js

  • Visual Studio Code

    Um editor de Texto/IDE
    Desenvolvido pelo Microsoft, o VSCode é bem completo, possui também versões pagas e mais completas

  • Notepad++

    Bem simples de usar
    De todos, notepad++ é o mais básico, mas nem por isso deixam de usá-lo. Quem gosta, usa e aprova.

Estrutura básica do HTML

Agora que você conhece a estrutura do HTML, vamos iniciar o primeiro projeto.

Repare que, comparado a estrutura inicial, foi adicionado a tag <p> </p>, com o texto "Olá, Mundo!" dentro.
A tag </p> define um novo parágrafo, com isso, colocamos um texto a ser exibido no navegador, dentro da tag.


Criando um novo arquivo:

File > New File


Mude o tipo de texto do arquivo:

Plain Text > HTML



Digite as tags, como no exemplo anterior:

Dentro do BODY, insira a tag <p> </p>, com o texto "Olá, Mundo" dentro

<p> Olá, Mundo! </p>


Salve o projeto com o nome "index"

File > Save As...

Salve o arquivo em alguma pasta ou local que quiser.


Abra o arquivo com seu navegador

Pode ser necessário clicar com o botão direito, depois "Abrir com..." e selecione seu navegador


É isso...
