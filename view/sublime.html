Baixando o Sublime Text

Utilizarei neste tutorial, o Sublime, mas fica a preferência de cada um, pois você pode usar qualquer editor de texto, incluindo o bloco de notas, do Windows.


Accesse o site do Sublime


Faça Download compatível com a sua versão de Sistema.

No meu caso, utilizo o Windows 64 Bit.

Você pode optar pela versão portable, no windows, como eu fiz. É bem prático e de fácil uso.


Extraia o arquivo .zip em alguma pasta

Caso você baixe o arquivo .exe, instale normalmente com o famoso "next, next and finish".


Execute o arquivo "sublime_text".

Sem segredos, dois cliques já basta, haha


Seja bem-vindo ao Sublime Text Portable

Depois de baixado, você poderá utilizá-lo para diversas linguagens, como: html, css, javascript, C, Java, C#

Lembrando que o Sublime é apenas um editor de texto, portanto, não é uma IDE para compilar o código.
